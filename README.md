# Quick Search Extension

## Description

This extension helps users to quickly search on their favorite websites.

## Installation

1. Install from the Chrome Web Store: [https://chromewebstore.google.com/detail/quick-search/lloohfpnnajcghiioiajbeimomlhfphg](https://chromewebstore.google.com/detail/quick-search/lloohfpnnajcghiioiajbeimomlhfphg)
2. Clone the repository and load it as an unpacked extension in Chrome Developer Mode:
   - Clone this repository.
   - Open Chrome and visit `chrome://extensions/`.
   - Enable `Developer mode` in the top right.
   - Click `Load unpacked` and select the cloned directory.

## Usage

After installation, click the extension icon to start searching.
